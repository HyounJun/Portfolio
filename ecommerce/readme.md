# 이커머스 고객 행동 분석 (2019년 12월)

온라인 쇼핑몰의 고객 행동 로그 **350만 건**을 분석하여 구매 이탈 원인을 파악하고 매출 증대를 위한 인사이트를 도출했습니다.

## 분석 구조

```
데이터 품질 확인 (EDA)
    ↓
구매 퍼널 분석 (조회 → 장바구니 → 구매)
    ↓
왜 이탈하는가?
    ├── 가설 1: 가격이 높을수록 전환율이 낮다
    └── 가설 2: 시간대에 따라 전환율이 다르다
```

---

## EDA 주요 발견

- **데이터 품질**: category_code 98.4% 결측, brand 42.7% 결측 확인
- **요일 패턴**: 평일(월~금) 이벤트가 주말 대비 평균 25% 이상 많음
- **가격 분포**: 전체 이벤트의 75%가 $7.14 이하 저가 상품에 집중

---

## 퍼널 분석

```
조회 (794,471)
    │  14.93% 전환
    ▼
장바구니 (118,646)  ← 85.07% 이탈
    │  15.82% 전환
    ▼
구매 (18,773)       ← 84.18% 이탈
```

두 단계 모두 이탈률이 84% 이상으로, **조회→장바구니 구간의 이탈이 더 큰 병목**으로 확인

---

## 가설 1: 가격이 높을수록 구매 전환율이 낮다 ✅ 검증됨

| 가격대 | 조회 수 | 조회→구매 전환율 | 조회→장바구니 전환율 |
|---|---|---|---|
| $0-5 | 725,939 | 3.42% | 11.43% |
| $5-10 | 377,523 | 2.68% | 10.00% |
| $10-20 | 193,103 | 2.31% | 8.11% |
| $20-50 | 110,850 | 1.53% | 5.67% |
| $50-100 | 51,627 | 1.01% | 3.99% |
| $100+ | 27,264 | 0.61% | 2.62% |

**가격이 오를수록 전환율이 하락 — 최저가($0-5) 대비 최고가($100+) 구간 전환율이 5.6배 차이**

---

## 가설 2: 시간대에 따라 구매 전환율이 다르다 ✅ 검증됨

- 최저 전환율: **1.92%** (0시)
- 최고 전환율: **3.25%** (10시)
- 전환율 차이: **1.33%p**

오전 8~11시에 전환율이 집중적으로 높게 나타남. 단, 가설 1의 가격 영향(5.6배)에 비해 상대적으로 작은 차이

---

## 인사이트 및 제언

1. **저가 상품 묶음 판매**: 전체 74%가 $10 이하에 집중 → 객단가 향상 전략 필요
2. **고가 상품 상세페이지 개선**: $100+ 구간은 조회(2.62%)에서부터 장벽 존재 → 상세 정보·리뷰 강화로 신뢰도 개선
3. **오전 집중 마케팅**: 8~11시 전환율 피크 구간에 광고·푸시 집중
4. **평일 마케팅 강화**: 평일 이벤트 수가 주말 대비 많아 평일 전환율 개선 효과가 큼
